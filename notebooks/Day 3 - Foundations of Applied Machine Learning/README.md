# Embedded Technical Skills Across Sessions

This package distributes the technical crash course across the existing workshop sessions.

## Session alignment

1. `01_session1_loading_and_framing.ipynb`
   - pandas loading and inspection
   - inputs, target, units, unit of observation

2. `02_session2_visualization.ipynb`
   - Seaborn and Matplotlib
   - scatterplot, histogram, and count plot

3. `03_session3_regression_sklearn.ipynb`
   - minimal scikit-learn regression workflow
   - mean baseline, linear regression, MAE, R²

## Requirements

```bash
pip install pandas numpy matplotlib seaborn scikit-learn jupyter
```
